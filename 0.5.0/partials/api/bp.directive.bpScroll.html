bpScroll
directive in module bp

Description

Makes the content within the directive scrollable via CSS.

Usage

as attribute
<ANY bp-scroll
     bp-scroll-sticky="{boolean}">
   ...
</ANY>

Parameters

ParamTypeDetails
bpScrollStickyboolean

Enables sticky headers for ios.

Example

<bp-table bp-scroll bp-scroll-sticky>
  <div>
    <bp-table-header>A</bp-table-header>
    <bp-cell>Aa</bp-cell>
    <bp-cell>Aa</bp-cell>
    <bp-cell>Aa</bp-cell>
    <bp-cell>Aa</bp-cell>
    <bp-cell>Aa</bp-cell>
  </div>
</bp-table>
