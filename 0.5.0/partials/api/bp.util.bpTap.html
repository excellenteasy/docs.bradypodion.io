bpTap
service in module bp.util

Description

bpTap is a service that is used to fire a tap event without 300ms click delay on an element.

Usage

bpTap(element[, attrs]);

Parameters

ParamTypeDetails
elementelement

The element that should receive the tap event.

attrs
(optional)
attrs

The element's attributes array as provided by the link function.

Returns

object

A tap object containing various information and references. Use tap.disable() on $destroy to prevent memory leaks.
