bpTab
directive in module bp

Description

A tab within a bpTabbar.

Dependencies

  • bpView

Usage

as element:
<bp-tab
       ui-sref="{string}"
       bp-tab-icon="{string}"
       [bp-tab-title="{string}"]>
</bp-tab>

Directive info

  • This directive creates new scope.
  • This directive executes at priority level 100.

Parameters

ParamTypeDetails
uiSrefstring

The name of the state the tab is associated to.

bpTabIconstring

A CSS class that represents a font-icon for the tab.

bpTabTitle
(optional)
string

The title of the tab. By default it's read from the associated state.

Example

<bp-tabbar>
  <bp-tab ui-sref='first'  bp-tab-icon="icon-bookmark"></bp-tab>
  <bp-tab ui-sref='second' bp-tab-icon="icon-comment"></bp-tab>
  <bp-tab ui-sref='third'  bp-tab-icon="icon-download"></bp-tab>
  <bp-tab ui-sref='fourth' bp-tab-icon="icon-inbox"></bp-tab>
  <bp-tab ui-sref='fifth'  bp-tab-icon="icon-music"></bp-tab>
</bp-tabbar>
