bpTap
directive in module bp

Description

bpTap is similiar to Angular's own ngClick directive, as it executes the provided expression, but without 300ms delay.

bpBoundMargin and bpNoScroll are automatically set based on context.

Bradypodion doesn't rely on ngTouch's click directive, as this one provides fine grained control.

You may define default configuration for taps with the provider.

Dependencies

  • bpTap

Usage

as attribute
<ANY bp-tap="{expression}"
     [bp-active-class="{string}"]
     [bp-allow-click="{boolean}"]
     [bp-bound-margin="{number}"]
     [bp-no-scroll="{boolean}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
bpTapexpression

Expression to evaluate upon tap.

bpActiveClass
(optional)
string

CSS class that is applied to the element during tap. (bp-active)

bpAllowClick
(optional)
boolean

Whether to allow the original click event or not. (false)

bpBoundMargin
(optional)
number

Threshold one is allowed to move the finger before the tap is cancelled. (50)

bpNoScroll
(optional)
boolean

Whether to allow scrolling during tap or not. (false)

Example

<bp-toolbar>
  <bp-action bp-tap="reload()">Reload</bp-action>
  <bp-action bp-tap="openShareSheet()">Share</bp-action>
</bp-toolbar>
