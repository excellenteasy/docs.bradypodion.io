bpSref
directive in module bp

Description

bpSref is similiar to Angular UI Router's own uiSref directive, as it transitions to a state, but without 300ms delay.

The tap behavior may be changed with the same attriubutes as with bpTap.

Dependencies

  • bpTap

Usage

as attribute
<ANY bp-sref="{string}">
   ...
</ANY>

Parameters

ParamTypeDetails
bpSrefstring

Name of the state to transition to. You can define params for the state like this: statename({id: 1}),

Example

<bp-navbar>
  <bp-action bp-sref="customers">Overview</bp-action>
  <bp-action bp-sref="customer({id: 4})">Detail</bp-action>
</bp-navbar>
