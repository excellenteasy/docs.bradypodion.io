bpActionOverflow
directive in module bp

Description

If there is no more space to display bpActions within a bpNavbar on android this menu is created. Normally you won't create it by yourself, but it will be created for you by the navbar. This is described in more detail in the bpNavbar documentation.

bpToolbar is the ios equivalent of bpActionOverflow.

Dependencies

  • bpApp

Usage

as element:
<bp-action-overflow>
</bp-action-overflow>

Example

<bp-action-overflow>
  <bp-action ng-click="doSomething()">First</bp-action>
  <bp-action ng-click="doSomething()">Second</bp-action>
  <bp-action ng-click="doSomething()">Third</bp-action>
  <bp-action ng-click="doSomething()">Fourth</bp-action>
  <bp-action ng-click="doSomething()">Fifth</bp-action>
</bp-action-overflow>
