bpNavigation
directive in module bp

Description

bpNavigation is a high-level abstraction for the entire app navigation. It assumes that every state should have a bpNavbar and automatically injects one into the screen for you.

bpNavigation handles transitions between navbars and and allows to configure the injected navbar via bpNavbarConfig.

Dependencies

  • bpNavigation

Usage

as attribute
<ANY bp-navigation>
   ...
</ANY>

Example

<body ng-app="bradypodionApp" bp-app bp-navigation>
  <ui-view-wrapper>
    <div ui-view></div>
  </ui-view-wrapper>
</body>
