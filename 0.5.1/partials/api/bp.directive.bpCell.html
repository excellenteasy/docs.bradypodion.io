bpCell
directive in module bp

Description

bpCell is a interface element that, together with other cells, constitutes a bpTable.

Usage

as element:
<bp-cell
       class="{string}">
</bp-cell>

Directive info

  • This directive executes at priority level 10.

Parameters

ParamTypeDetails
classstring

You can define different cell styles.

bp-cell-value1 | bp-cell-value2 | bp-cell-subtitle

Example

<bp-cell>Cell</bp-cell>

<bp-cell class="bp-cell-value1">
  <div class="label">Label</div>
  <div class="value">Value</div>
</bp-cell>

<bp-cell class="bp-cell-value2">
  <div class="label">Label</div>
  <div class="value">Value</div>
</bp-cell>

<bp-cell class="bp-cell-subtitle">
  <div class="titles">
    <div class="title">Title</div>
    <div class="subtitle">Subtitle</div>
  </div>
</bp-cell>
