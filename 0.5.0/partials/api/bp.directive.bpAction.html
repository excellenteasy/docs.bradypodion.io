bpAction
directive in module bp

Description

bpAction is a general type of interface element that is mainly used within bpNavbar. It supports bpButton and bpIcon types. The abstraction is necessary as in some occasions an action is an icon on android, while it's a button on ios. This is described in more detail in the bpNavbar documentation.

Usage

as element:
<bp-action
       class="{string}">
</bp-action>

Parameters

ParamTypeDetails
classstring

Add either the bp-button or bp-icon class to an action element.
