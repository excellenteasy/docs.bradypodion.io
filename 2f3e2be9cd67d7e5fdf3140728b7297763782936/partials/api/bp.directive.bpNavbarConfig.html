bpNavbarConfig
directive in module bp

Description

bpNavbarConfig is a way to configure navbars injected by the bpNavigation directive. It offers exactly the same attributes for configuration as the original navbar, and forwards them to it. If you put bpActions into the directive they will be moved to the original navbar.

Dependencies

  • bpNavigation

Usage

as element:
<bp-navbar-config>
</bp-navbar-config>

Example

<bp-navbar-config bp-navbar-no-up>
  <bp-action class="icon-checkmark">
    Done
  </bp-action>
</bp-navbar-config>
