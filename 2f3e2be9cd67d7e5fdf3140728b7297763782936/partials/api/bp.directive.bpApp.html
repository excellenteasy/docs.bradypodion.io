bpApp
directive in module bp

Description

The bpApp directive is similiar to Angular's own ngApp directive. It is used to initalize a Bradypodion application, as it applies important CSS information and sets up transitions. You will be fine by putting it right next to the ngApp directive.

Dependencies

  • bpApp
  • bpView

Usage

as element:
<bp-app>
</bp-app>
as attribute
<ANY bp-app>
   ...
</ANY>

Example

<html>
  <head>
    <title>Bradypodion App</title>
  </head>
  <body ng-app="bradypodionApp" bp-app>
    ...
  </body>
