bpIscroll
directive in module bp.iscroll

Description

Makes the content within the directive scrollable via iScroll.

Dependencies

  • bpApp
  • bpIscroll

Usage

as attribute
<ANY bp-iscroll
     [bp-iscroll-sticky="{string}"]>
   ...
</ANY>

Parameters

ParamTypeDetails
bpIscrollSticky
(optional)
string

Enables sticky headers for ios. You can define the CSS Selector that is used to determine what a header is (bp-table-header).

Example

<bp-table bp-iscroll bp-iscroll-sticky>
  <div>
    <bp-table-header>A</bp-table-header>
    <bp-cell>Aa</bp-cell>
    <bp-cell>Aa</bp-cell>
    <bp-cell>Aa</bp-cell>
    <bp-cell>Aa</bp-cell>
    <bp-cell>Aa</bp-cell>
  </div>
</bp-table>
